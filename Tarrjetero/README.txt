Para compilar la practica:

Primero debemos instalar pymysql para poder usas SQL Server

despues de eso es necesario que para todas las consultas se ponga en nombre de la 
base de datos, en mi caso la base de datos se llama "mydb" y para tambien es necesario
el nombre del usuario y la contraseña, en mi caso accedo con rool y la contraseña es:
pruebatest

se puede descargar mySql server desde la pagina oficial

despues de descargar la herramienta lo que sigue es ejecutar el script que esta en la carpeta de
"Base de datos"

Y por ultimo es necesario situarse en la carpeta principal y ejecutar el sigueinte codigo:

python3 main.py
